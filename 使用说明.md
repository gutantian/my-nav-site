# 导航站使用说明

## 如何编辑网站信息

现在你可以通过编辑 `websites.js` 文件来方便地管理所有网站信息，无需修改 HTML 代码。

### 编辑步骤

1. **打开配置文件**：用文本编辑器打开 `websites.js` 文件
2. **修改网站信息**：按照以下格式编辑网站数据
3. **保存文件**：保存后刷新浏览器即可看到更新

### 配置文件格式

```javascript
const websites = {
  分类名称: [
    {
      name: "网站名称",
      url: "网站地址",
      desc: "网站描述",
      icon: "可选的图标表情"
    },
    // 更多网站...
  ],
  // 更多分类...
};
```

### 示例：添加新网站

```javascript
// 在 AI 工具分类中添加新网站
av: [
  {
    name: "ChatGPT",
    url: "https://chat.openai.com",
    desc: "OpenAI 出品的对话 AI",
    icon: "🤖"
  },
  {
    name: "新网站",  // 新增网站
    url: "https://example.com",
    desc: "网站描述",
    icon: "🌟"
  }
]
```

### 示例：修改现有网站

```javascript
// 修改 GitHub 网站信息
dev: [
  {
    name: "GitHub",
    url: "https://github.com",  // 修改为新地址
    desc: "全球最大代码托管平台",  // 修改描述
    icon: "🐙"
  },
  // 其他网站...
]
```

### 示例：删除网站

直接删除对应的网站对象即可：

```javascript
dev: [
  {
    name: "GitHub",
    url: "https://github.com",
    desc: "全球最大代码托管平台",
    icon: "🐙"
  },
  // 删除了 Stack Overflow
  {
    name: "MDN Web Docs",
    url: "https://developer.mozilla.org/zh-CN/",
    desc: "最权威的 Web 技术文档",
    icon: "📖"
  }
]
```

## 如何添加新分类

1. **在配置文件中添加分类**：在 `websites` 对象中添加新的分类
2. **在 HTML 中添加分类**：在侧边栏和主内容中添加对应的分类结构

### 示例：添加新分类

1. 在 `websites.js` 中添加：
```javascript
const websites = {
  // 现有分类...
  newCategory: [
    {
      name: "网站1",
      url: "https://example1.com",
      desc: "描述1",
      icon: "📌"
    }
  ]
};
```

2. 在 `index.html` 侧边栏中添加：
```html
<a class="nav-link" onclick="scrollToSection('newCategory')">
  <span class="icon">📌</span> 新分类
</a>
```

3. 在 `index.html` 主内容中添加：
```html
<!-- 新分类 -->
<div class="section" id="sec-newCategory">
  <div class="section-header">
    <span class="section-icon">📌</span>
    <span class="section-title">新分类</span>
    <span class="section-count">0</span>
  </div>
  <div class="cards"></div>
</div>
```

4. 在 `index.html` 的 `scrollToSection` 函数中添加映射：
```javascript
const map = { 
  ai:'sec-ai', 
  dev:'sec-dev', 
  design:'sec-design', 
  info:'sec-info', 
  productivity:'sec-productivity', 
  learn:'sec-learn',
  newCategory:'sec-newCategory'  // 新增映射
};
```

5. 在 `renderWebsites` 函数中添加映射：
```javascript
const sectionMap = {
  ai: 'sec-ai',
  dev: 'sec-dev',
  design: 'sec-design',
  info: 'sec-info',
  productivity: 'sec-productivity',
  learn: 'sec-learn',
  newCategory: 'sec-newCategory'  // 新增映射
};
```

## 本地预览

在项目目录中运行：
```bash
python -m http.server 8000
```

然后在浏览器中访问：`http://localhost:8000`

## 注意事项

- 确保 JSON 格式正确，每个网站对象之间用逗号分隔
- 图标使用表情符号，确保浏览器支持显示
- 网站地址必须包含 `http://` 或 `https://`
- 修改后需要刷新浏览器才能看到更新

现在你可以轻松管理你的导航站了！🎉
